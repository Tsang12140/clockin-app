# 安卓保存工资条图片到相册 — HTTPS 升级指南

## 现状

当前服务器使用 HTTP。`navigator.share`（Web Share API）强制要求 HTTPS，所以目前只能下载到"下载"文件夹，无法直接保存到相册。

HTTPS 上线后，代码无需改动，保存图片会自动弹出系统分享菜单，选"保存图片"即存入相册。

---

## 步骤一：申请免费 SSL 证书（宝塔面板）

1. 登录宝塔面板 → **网站** → 找到 clockin 站点 → **设置**
2. 左侧菜单选 **SSL** → **Let's Encrypt**
3. 勾选你的域名（确保域名 DNS 已解析到服务器 IP）
4. 点击**申请** → 等待约 30 秒 → 开启**强制 HTTPS**
5. 宝塔会自动续签，有效期 90 天

---

## 步骤二：更新 iron-session 配置

申请 SSL 成功后，找到项目中的 session 配置文件，将 `secure: false` 改为 `secure: true`：

```typescript
// 通常在 lib/session.ts 或 app/login/actions.ts 附近
export const sessionOptions = {
  password: process.env.SESSION_SECRET!,
  cookieName: 'clockin-session',
  cookieOptions: {
    secure: true,   // ← 改这里（原来是 false）
    httpOnly: true,
  },
};
```

---

## 步骤三：重新部署

```bash
cd /www/wwwroot/clockin-app
npm run build
pm2 restart clockin
```

---

## 验证

1. 浏览器访问 `https://你的域名`，地址栏出现绿色锁 ✓
2. 打开任意员工工资条 → 点"保存图片"
3. 安卓弹出系统分享菜单 → 选"保存到相册"或"保存图片" ✓

---

## 注意事项

- 如果域名没有备案，阿里云国内服务器 80/443 端口可能被封，需先备案
- 如果 Let's Encrypt 申请失败，检查域名是否正确解析到服务器 IP（`ping 你的域名`）
- 更改 `secure: true` 后，旧的 HTTP Cookie 会失效，用户需要重新登录一次
